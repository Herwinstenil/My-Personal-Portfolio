Automated XML Proofing Tool

I built an Automated XML Proofing Tool to streamline document comparison and validation, enabling side-by-side XML comparison with highlighted differences. I implemented advanced proofing features including duplicate detection, gibberish removal, and footnote handling, along with a Flask backend integrated with MongoDB to manage comparison results and track user actions efficiently.

  • Flask
  • MongoDB
  • JavaScript (ES6+)
